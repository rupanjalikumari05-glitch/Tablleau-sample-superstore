# Sales Performance Dashboard – Sample Superstore

## Project Overview
This project analyzes sales and profit data using Tableau on the Sample Superstore dataset.

## Tools Used
- Tableau
- Data Visualization

## Dashboard Components
- Total Sales & Total Profit
- Monthly Sales Trend
- Region-wise Profit
- Top 10 Sub-Categories
- Loss Making Products

## Key Insights
- West region is the most profitable.
- Sales peak during November–December.
- Tables sub-category causes maximum loss.
- Phones and Chairs are top-selling products.
